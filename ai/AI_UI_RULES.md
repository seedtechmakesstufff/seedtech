# UI Design Rules

Avoid generic AI landing page layouts.

Use:

large typography
generous spacing
layered sections
subtle gradients
card UI blocks with glass effect
hover animations
visual hierarchy

 STYLE: Soft UI Evolution
>Keywords: Soft shadows, subtle depth, premium feel, 
>Best For: Web Dev & Design, IT Agency, premium services
>Performance: Excellent | Accessibility: WCAG AA      

Components should feel premium and modern.
