# AI Development Rules

All code must use:

Next.js
React
TypeScript
TailwindCSS
ShadCN UI
neon via vercel integration

Rules:

Use reusable components.

Do not place large layouts inside pages.

Pages should import components from /components.

Components should remain under 200 lines.

Use Tailwind spacing scale.

Avoid generic Tailwind layouts.

Design should feel premium and modern.

All pages must be mobile responsive.
